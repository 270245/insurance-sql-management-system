# Insurance Management Database (SQL Project)

## Overview
This project simulates an insurance system used to manage customers, policies, claims, and payments.

## Skills Demonstrated
- SQL (T-SQL)
- Database design
- Joins and relationships
- Data analysis queries

## Tables
- Customers
- Policies
- Claims
- Payments

## How to Use
1. Run schema.sql
2. Run sample_data.sql
3. Run queries.sql
